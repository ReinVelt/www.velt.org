üß© PUZZLE SOLUTIONS

Puzzle 1: ROT1 Cipher (SSTV Decryption)

üîê Caesar Cipher - Shift of 1

Location

Mancave ‚Üí Laptop ‚Üí Decrypt SSTV Message

Encrypted Message

XBSOJOH - QSPKFDU FDIP JT DPNQSPNJTFE - NPWF UP CBDLVQ DIBOOFM - DPPSEJOBUFT GPMMPX - USVTU OP POF

How to Solve

ROT1 cipher means each letter is shifted back by 1 position in the alphabet:

  • A ‚Üí Z
  • B ‚Üí A
  • C ‚Üí B
  • etc.

Decrypted Message

WARNING - PROJECT ECHO IS COMPROMISED - MOVE TO BACKUP CHANNEL - COORDINATES FOLLOW - TRUST NO ONE

In-Game Solution

The game provides a decryption interface on the laptop. Players must recognize it's a simple shift cipher and decode it.

Skip ROT1 Puzzle: game.setFlag('sstv_decoded', true); game.setFlag('klooster_unlocked', true);
