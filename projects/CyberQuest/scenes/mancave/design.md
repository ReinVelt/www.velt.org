# Scene: Mancave (Garage)

## Overview
Ryan's tech lab/workshop next to the house. Packed with hacking equipment, electronics tools, and the crucial SSTV terminal that receives the mysterious transmission.

## Visual Description
- Cramped garage converted to tech lab
- Dim lighting with LED displays glowing
- Multiple screens and monitors
- Workbenches covered with equipment
- Organized chaos of wires and components

## Equipment (Hotspots)

### 1. SSTV Terminal
- The key item that receives the mysterious transmission
- Initially shows only static/noise
- After Story Part 2 trigger, shows the morse code pattern

### 2. Laptop
- Main computer for coding and analysis
- Used to decode messages
- Shows email, code editor

### 3. HackRF One with Portapack
- Software Defined Radio
- Used to tune into military frequency
- Puzzle: Frequency tuner

### 4. Flipper Zero
- Multi-tool for RFID/NFC
- Can be added to inventory

### 5. 3D Printer
- Background equipment
- Examine action

### 6. Laser Cutter
- Background equipment
- Examine action  

### 7. Server Rack
- Terminal access
- Dead man's switch setup

### 8. Soldering Station
- Workbench area
- Tools for electronics

### 9. Door to House
- Navigate back to kitchen

## Story Events
- Story Part 2: SSTV transmission begins
- Story Part 3: Message decoding puzzle (ROT1)

## Puzzles
