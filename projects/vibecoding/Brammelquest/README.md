## About

You are **Bram**, a lovable pug (also known as "vies ventje") on an epic quest to find the legendary **Holy Sausage**! This game is inspired by classic Sierra adventure games like "Leisure Suit Larry" and "King's Quest".

## Story

Bram wakes up one morning to discover that the Holy Sausage - a legendary artifact said to grant infinite belly rubs - has been stolen! With nothing but his keen nose and determination, Bram must sniff out clues, talk to suspicious characters, and solve puzzles to recover the sacred sausage.

## How to Play

### Controls

| Key | Action |
|-----|--------|
| **Click** | Perform current action |
| **W** | Walk mode |
| **L** | Look at things |
| **U** | Use/interact with objects |
| **T** | Talk to characters |
| **I** | Open inventory |
| **ESC** | Close dialogs/inventory |

### Gameplay

1. **Explore** - Click around scenes to move Bram
2. **Look** - Examine objects and characters for clues
3. **Talk** - Have conversations with NPCs
4. **Use** - Interact with objects and pick up items
5. **Inventory** - Use collected items to solve puzzles

## Running the Game

Simply open `index.html` in a modern web browser, or run a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx http-server

# Using PHP
php -S localhost:8000
```

Then visit `http://localhost:8000`
