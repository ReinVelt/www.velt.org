# Regenwormen/Pickomino Learning Game

## Project Overview

This project is an educational implementation of the Regenwormen (Pickomino) board game, created as part of a learning process to understand game mechanics and algorithms before tackling a larger assignment.

## Background & Purpose

**Author**: Rein Velt  
**Context**: Assignment preparation for Dawn Technology  
**Objective**: Understanding Regenwormen/Pickomino game mechanics and logic

### Why This Project Exists

I received an assignment from Dawn Technology involving Regenwormen/Pickomino algorithms. Since I was unfamiliar with the game rules and strategy, I needed a hands-on approach to understand the problem space before developing efficient algorithms.

This implementation serves as:
- **Step 1**: Problem comprehension and game mechanics exploration
- **Learning Tool**: Interactive way to understand game flow, decision points, and strategic elements
- **Foundation**: Base knowledge for approaching the actual assignment requirements
