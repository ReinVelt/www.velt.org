Ledder is a framework for creating pixel animations on LED matrices, developed by psy0rz. Within this project, I've contributed "Rein's Collection" â€“ a collection of more than 40 animations and experiments that explore the possibilities of the framework.

About Ledder

Ledder enables you to create real-time animated pixel art and stream it to LED matrix displays. Through a web interface, you can control and adjust animations in real-time. The project supports various hardware drivers and output methods, including web browser displays, video streaming, and physical LED panels (such as Ws2812B, Hub75 and more experemental like Colorlight-5A75B FPGA controller and Chromecast).

Ledder can work on a RaspBerry Pi, a Docker instance, or bare metal: 'npm run build' and 'npm run start' will do the startup magic.

Live demo: https://ledder.datux.nl
GitHub: https://github.com/psy0rz/ledder

Philosophy

This collection is meant as inspiration and a learning resource. The goal is to showcase what's possible with psy0rz's Ledder framework, not necessarily to provide production-ready code.

As stated in the collection's README:

"These examples are just proof-of-concepts to show how easy it is to create a ledder animation. Some animations are really nice, but some suck and will steal your computers processortime and memory to create ten-thousands of obsolete pixels. You can use this code for your own animations under the condition that you (try to) make things more pretty than i did ;-)"
