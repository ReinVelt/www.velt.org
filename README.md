# velt.org

My personal website repository.

## Getting Started

This repository is set up and ready for your website code!

### Importing Your Code from Server

If you have code on your server that you want to import into this repository, please see the comprehensive [Import Guide](IMPORT_GUIDE.md) for step-by-step instructions.

The guide covers multiple methods:
- Direct Git push from your server
- Download and upload approach
- Using GitHub web interface
- Using GitHub Desktop

### Quick Start for New Code

If you're starting fresh:

1. Clone this repository:
   ```bash
   git clone https://github.com/ReinVelt/velt.org.git
   cd velt.org
   ```

2. Add your code files

3. Commit and push:
   ```bash
   git add .
   git commit -m "Add website code"
   git push origin main
   ```
